# 몬스터 벽돌깨기 히어로 — Prototype

이 디렉토리는 순수 HTML/CSS/JS로 작성된 벽돌 깨기(Breakout) 게임의 최소 뼈대입니다.

파일 구조

- `index.html` — 진입점 (canvas 포함)
- `css/styles.css` — 기본 스타일
- `js/game.js` — 게임 로직(그리기, 물리, 입력). 현재 버전은 "턴" 기반으로 동작하며, 한 턴에 10개의 공을 발사합니다.
- `js/game.js` — 게임 로직(그리기, 물리, 입력). 현재 버전은 "턴" 기반으로 동작하며, 한 턴에 10개의 공을 발사합니다.
    - 턴 시작 후 마우스로 발사 각도를 실시간으로 선택하고, 캔버스 클릭으로 선택한 각도로 모든 공을 순차 발사합니다.
    - 벽돌 배치는 랜덤으로 일부 빈 칸을 포함하며, 3턴이 진행될 때마다 벽돌이 한 칸씩 아래로 내려옵니다.

- 빠른 실행(Windows PowerShell)
- 단순히 로컬에서 열기:

```powershell
# 파일 탐색기에서 열기
Start-Process "D:\Dev\Workspace\AI\monster-breakout-hero\index.html"
```

- 간단한 로컬 서버(권장, CORS/리소스 로딩 안정성 확보):

```powershell
cd D:\Dev\Workspace\AI\monster-breakout-hero
python -m http.server 8000
# 그런 다음 브라우저에서 http://localhost:8000 를 엽니다
```

다음 단계(권장)

- 오디오(타격/클리어 효과) 추가
- 브릭 종류(몬스터 속성), 애니메이션, 레벨 시스템 추가
- 모바일 터치/제스처 입력 지원
- 게임 규칙 변경: 현재는 턴당 10개의 공을 발사하여 브릭을 깨는 방식입니다. 공을 놓치더라도 목숨이 줄어들지 않고 턴이 끝나면 다음 턴을 시작할 수 있습니다.
